#  YouTube UI Clone

##  Overview
This project is a **frontend YouTube UI clone** built using HTML and CSS. It replicates the basic layout of YouTube, including the header, sidebar, and video grid.

##  Features
-  Video grid layout (YouTube-style)
-  Real thumbnails with proper scaling
-  Channel icons and video details
-  Responsive design
 - Modern UI with hover effects


##  Tech Stack
  HTML  
  CSS
